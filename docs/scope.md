## Project Scope Statement: Quality Time with Close Family and Friends
### Project Name: Liz's Familia & Amigos Circle
#### Project Manager: Lizeth Arellano
### Project Purpose
This project is a simple website to plan and execute social gathering with
immediate family and friends to strengthen relationships. This project will focus
on creating organized activities like monthly dinners, weekend hangouts, and even
game nights. As far as feedback loops, family and friends can input activity
preferences before each event via chat. This is to keep in touch with family and
friends between our busy lives.
### Description
This project will be for planning and executing social gatherings with family and
friends ranging from simple at home game nights with friends to dinners at
restaurants. This will help with keeping track of dates of gatherings and with
whom. The key team members is Lizeth Arellano as the project manager and a cousin
that can help with the planning.
### Desired Results
One specific outcome that I hope to achieve is the completion of at least one
gathering per month with my best friends and one with my cousins.
I would prioritize quality as 1, time as 2, and budget as 3.
* Time: This project is due March 1, 2026, for the end of the semester.
* Quality: "Done" will look like a completion of at least 2 planned events for the
next 2 months with family members.
* Budget: The budget can be $0 from a simple hangout at either participants home,
but can go up depending on the restaurant dinner pricing. This project is simply to
plan to get to see each other more often.
### **Website Requirements**
* index.html (homepage)
* about.html (secondary page)
* style.css (external stylesheet)
### Deliverables
* A GitHub repository dedicated to all project documentation and event data.
* All plan and budget updates will be input through commits in GitHub.
* Website with activity logs and feedback forms.
### Exclusions
Absolutely no large scale events like family reunions. This project is for
immediate family and very close friends only. Also, no high cost vacations or
international travel. No events that require overnight stays. The final sign off
will be from the project manager and immediate family and close friends. There will
be constant sign offs for attendance to events planned as well.
