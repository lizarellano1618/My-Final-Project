## Project Plan: Liz's Familia & Amigos Circle
This follows the guidelines from Chapter 4 of PMUPM to connect with family at least twice a month.
### 1. Simple Task Schedule
 - Setup: Create a GitHub Repository
 - Code: Write required md, html, and css files
 - Docs: Create docs/ folder with required md files
 - Launch: Enable required GitHub pages
 ### 2. Risk Analysis (TAME)
- Transfer: Use GitHub to host the site
- Avoid: Security risks and complex bugs
- Mitigate: Use simple design so the site is easy to follow
- Embrace: Post a summary of the event in case anyone missed the event
### 3. Goals
- Have two events scheduled per month
- Have a detailed homepage in regards to the site's purpose
- Ensure the site works via GitHub
